## 📋 Sobre o Projeto

Este site foi desenvolvido como parte do projeto avaliativo do 1º trimestre, com o tema **Araucárias e a Cultura do Paraná**. O objetivo é promover a conscientização sobre a importância da preservação da Araucária (Araucaria angustifolia) e celebrar as tradições culturais paranaenses.

### 🎯 Objetivos do Projeto

- Divulgar informações sobre a araucária e sua importância ecológica
- Valorizar a cultura paranaense (culinária, festas, tradições)
- Aplicar conhecimentos de HTML, CSS e JavaScript
- Criar uma experiência interativa e responsiva
- Demonstrar criatividade e originalidade na releitura do tema
